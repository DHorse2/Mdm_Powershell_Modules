Mdm_Bootstrap

Type: Function

Synopsis: Setup (bootstrap) Windows for the Development Environment.

Detailed Description: Text: This is the first step of bootstrapping the Development Environment. This updates the Windows Environment variables. It installs these powershell modules to the system's directories. Set registry, Path and load PowerShell modules. $source = "$scriptPath\" $destination = "$Env:ProgramFiles\WindowsPowerShell\Modules"

Syntax: syntaxItem: @{name=Initialize-Dev_Env_Win; parameter=System.Management.Automation.PSObject[]}

Parameters:

- UpdatePath: Text: Switch: A switch to indicate the path should be checked/updated.

- DoPause: Text: Switch: Pause between steps.

- DoVerbose: Text: Switch: Verbose output and prompts.

Return: returnValue: @{type=@{name=todo.}}

Examples:

Example: -------------------------- EXAMPLE 1 -------------------------- Introduction: @{Text=PS > }       Syntax: Initialize-Dev_Env_Win -SilentMode -UpdatePath      Remarks:

Parameters:
