Name: Initialize-Dev_Env_Win about_Mdm_Dev_Env_Install

Type: Function HelpFile

Synopsis: Setup (bootstrap) Windows for the Development Environment.

Detailed Description: This is the first step of bootstrapping the Development Environment. This updates the Windows Environment variables. It installs these powershell modules to the system's directories.
