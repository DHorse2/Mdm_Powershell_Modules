Mdm_Std_Library

Type: Function

Synopsis: Generates the extended help files for the Mdm Modules.

Detailed Description: Text: This runs Get-Modules and Get-Command functions to create help files. The general intent is to have function lists. To correctly run this command (showing which module it belongs to) enter: ```powershell Import-Module -name Mdm_Std_Library -force ``` This function re-imports the other module in the correct order. Note: It should not need to be run unless changes were made to the Mdm Modules.

Syntax: syntaxItem: @{name=Write-Mdm_Help; parameter=System.Management.Automation.PSObject[]}

Parameters:

- moduleRoot: N/A

- DoPause: Text: Switch: Pause between steps.

- DoVerbose: Text: Switch: Verbose output and prompts.

- DoDebug: Text: Switch: Debug this script.

Examples:

Example: -------------------------- EXAMPLE 1 -------------------------- Introduction: @{Text=PS > }       Syntax: Import-Module -name Mdm_Std_Library -force
Write-Mdm_Help -DoVerbose -DoPause      Remarks:
